# Lab 4 — Maven

**OOP Java: Класи, ArrayList, Maven**

## Опис

Програма реалізує клас `Employee` (працівник) та драйвер-клас `Main`, який:
- зчитує з клавіатури кількість працівників;
- для кожного запитує ім'я та прізвище;
- зберігає всі об'єкти в `ArrayList<Employee>`;
- виводить інформацію через `toString()`;
- демонструє роботу методу `equals()`.

## Структура проєкту

```
employee-app/
├── pom.xml
├── .gitignore
├── README.md
└── src/
    └── main/
        └── java/
            └── com/
                └── lab1/
                    ├── Employee.java   ← клас предметної області
                    └── Main.java       ← драйвер (main)
```

## Поля класу Employee

| Поле              | Тип      | Опис               |
|-------------------|----------|--------------------|
| `firstName`       | `String` | Ім'я               |
| `lastName`        | `String` | Прізвище           |
| `position`        | `String` | Посада             |
| `salary`          | `double` | Зарплата (грн)     |
| `experienceYears` | `int`    | Стаж (роки)        |

## Збірка та запуск

```bash
# Збірка
mvn clean compile

# Запуск
mvn exec:java -Dexec.mainClass="com.lab.Main"
```
